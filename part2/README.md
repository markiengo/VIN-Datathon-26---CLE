# Part 2 — Analytics

Exploratory analysis of the ecommerce dataset: demand baseline, promo economics, returns & cancellations, category portfolio, and customer cohort retention.

## Files

| File | Description |
|---|---|
| `part2_analytics.ipynb` | Full analytics notebook with charts and commentary |
| `build_part2_analytics_notebook.py` | Script that programmatically builds the analytics notebook |

## How to run

```bash
# Option A: open the notebook directly
jupyter notebook part2/part2_analytics.ipynb

# Option B: regenerate the notebook from the builder script
python part2/build_part2_analytics_notebook.py
```

Report figures are generated by `scripts/build_report_assets.py` at repo root and saved to `report/assets/`.
