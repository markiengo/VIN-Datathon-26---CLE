# Part 1 — MCQ & Data Validation

This folder contains the multiple-choice answers and data validation work for Round 1.

## Files

| File | Description |
|---|---|
| `part1_mcq.ipynb` | MCQ answers with reasoning |
| `part1_data_validation.ipynb` | Data quality checks: row counts, date coverage, nulls, type assertions |

## How to run

Open either notebook from the repo root (Jupyter will set `cwd` to repo root, allowing `dataset/` to resolve correctly).
